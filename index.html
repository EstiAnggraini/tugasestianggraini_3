JENIS-JENIS CONSTRUCTOR

Dart menyediakan beberapa mekanisme inisialisasi objek untuk berbagai kebutuhan arsitektur:

  1. Default Constructor: ClassName()
  2. Named Constructor: ClassName.name()
  3. Parameterized Constructor: ClassName(this.param)
  4. Constant Constructor: const ClassName()
  5. Factory Constructor: (Akan dibahas pada sesi lanjut)

2. NAMED CONSTRUCTOR - CONTOH DETAIL

Dart memungkinkan pembuatan lebih dari satu konstruktor dalam satu kelas menggunakan Named Constructor. Ini sangat berguna untuk memberikan cara alternatif dalam menginisialisasi objek, seperti dari data JSON. 

// Nama: Esti Anggraini
// Nim: 251410031
// Kelas: SI2A

import 'dart:convert';

class Motor {
  String? merek;
  int? tahunProduksi;

  // Parameterized constructor
  Motor(this.merek, this.tahunProduksi);

  // Named constructor
 Motor.standar() {
    merek = 'Honda';
    tahunProduksi = 2023;
  }

  // Named constructor dengan parameter
  Motor.dariData(Map data) {
    merek = data['merek'];
    tahunProduksi = data['tahunProduksi'];
  }

  // Override toString for better print output
  @override
  String toString() {
    return 'Motor{merek: $merek, tahunProduksi: $tahunProduksi}';
  }
}

void main() {
  print('--- PRAKTIKUM KELAS MOTOR ---');

  // 1. Constructor default
  var motor1 = Motor('Yamaha', 2022);
  print('Motor 1 (constructor default): $motor1');

  // 2. Named constructor .standar
  var motor2 = Motor.standar();
  print('Motor 2 (named constructor .standar): $motor2');

  // 3. Named constructor .dariData
  String dataJsonMotor = '{"merek": "Kawasaki", "tahunProduksi": 2023}';
  Map jsonMotor = jsonDecode(dataJsonMotor);
  var motor3 = Motor.dariData(jsonMotor);
  print('Motor 3 (named constructor .dariData): $motor3');

  // 4. Contoh lain dengan .dariData
  Map dataMotorLain = {'merek': 'Ducati', 'tahunProduksi': 2024};
  var motor4 = Motor.dariData(dataMotorLain);
  print('Motor 4 (named constructor .dariData): $motor4');

  print('\n--- SELESAI DEMONSTRASI ---');
}

3. Constant Constructor

Konstruktor konstan digunakan untuk membuat objek yang immutable (tidak bisa diubah). Objek yang dibuat dengan const dan nilai yang sama akan menggunakan instance yang sama di memori, sehingga lebih efisien. 

// Nama: Esti Anggraini
// Nim: 251410031
// Kelas: SI2A

class WarnaCat {
  final int red, green, blue;
  const WarnaCat(this.red, this.green, this.blue);

  @override
  String toString() {
    return 'WarnaCat(red: $red, green: $green, blue: $blue)';
  }
}

void main() {
  const merahBata = WarnaCat(180, 60, 60);
  const merahMuda = WarnaCat(255, 180, 180); 

  print(merahBata);
  print(merahMuda); 
  print('Apakah objek sama? ${identical(merahBata, merahMuda)}'); 
  
  // Contoh tambahan untuk menunjukkan konsep canonical instance
  const biruLangit = WarnaCat(135, 206, 235);
  const biruLangitLagi = WarnaCat(135, 206, 235);
  
  print('\n--- Demonstrasi Canonical Instance ---');
  print(biruLangit);
  print(biruLangitLagi);
  print('Apakah biruLangit dan biruLangitLagi instance sama? ${identical(biruLangit, biruLangitLagi)}');
}

PENGANTAR ENKAPSULASI

Konsep Enkapsulasi

Konsep: Menyembunyikan detail implementasi dari luar.

Tujuan:

  • Keamanan data
  • Kontrol akses
  • Fleksibilitas perubahan

Cara di Dart: Menggunakan underscore (_) untuk private members.

Public: Dapat diakses dari mana saja (default).

Private: Hanya dapat diakses dalam file yang sama.

Getter And Setter

Pengertian Getter dan Setter

Getter: Memungkinkan membaca data private dengan kontrol.

Sintaks 1: tipeData get namaGetter => _field; (Arrow)

Sintaks 2: tipeData namaGetter() { return _field; } (Block)

Setter: Memungkinkan modifikasi data dengan validasi.

Sintaks 1: set namaSetter(value) {...} (Set Keyword)

Sintaks 2: void namaSetter(value) {...} (Method Approach)

// Nama: Esti Anggraini
// Nim: 251410031
// Kelas: SI2A 

class Person {
  String _nama1;
  int umur2;
  
  Person(this._nama1, this.umur2);
  
  set setNama(String nama) {
    _nama1 = nama;
  }

  void ubahNama(String nama) {
    _nama1 = nama;
  }

  String getNama() {
    return _nama1;
  }
}

void main() {
  Person p1 = Person("Esti Anggraini", 19);
  print('Umur: ${p1.umur2}');
  
  Person p3 = Person("", 0);
  p3.ubahNama('Anggraini');
  print('Nama p3 (via Method): ${p3.getNama()}');
  
  p1.setNama = 'Esti Anggraini';
  print('Nama (via Setter): ${p1.getNama()}');
  
  print('Nama akhir p1: ${p1.getNama()}');
}

// Nama: Esti Anggraini
// Nim: 251410031
// Kelas: SI2A

class Produk {
  String _nama;
  double _harga;
  
  Produk(this._nama, this._harga);
  
  String get nama => _nama;
  double get harga => _harga;
  
  set nama(String value) {
    if (value.length >= 3) {
      _nama = value;
    } else {
      print('Gagal memperbarui nama: Nama harus memiliki minimal 3 karakter.');
    }
  }
  
  set harga(double value) {
    if (value > 0) {
      _harga = value;
    } else {
      print('Gagal memperbarui harga: Harga harus lebih besar dari 0.');
    }
  }
}

void main() {
  var px = Produk('Asus SmartPanel', 3500.0);

  print('--- Informasi Produk Awal ---');
  print('Nama Produk: ${px.nama}');
  print('Harga Produk: Rp ${px.harga}');

  print('\n--- Memperbarui Produk ---');
  px.nama = 'Sonar Panel';
  px.harga = 12000.00;
  print('Nama (setelah update): ${px.nama}');

  print('\n--- Menguji Validasi Setter ---');
  px.nama = 'Termometer'; // Gagal (kurang dari 11 karakter)
  px.harga = -1000.02; // Gagal (kurang dari 0)
}

class RekeningBank {
  String _nomorRekening;
  double _saldo;

  RekeningBank(this._nomorRekening, this._saldo);

  void deposit(double jml) {
    if (jml > 0) {
      _saldo += jml;
      print('Deposit Rp $jml Berhasil.');
    }
  }

  void tarik(double jml) {
    if (jml > 0 && _saldo >= jml) {
      _saldo -= jml;
      print('Tarik Rp $jml Berhasil.');
    } else {
      print('Gagal: Saldo tidak cukup atau input tidak valid.');
    }
  }

  void cekSaldo() {
    print('Rekening: $_nomorRekening | Saldo: Rp $_saldo');
  }
}

void main() {
  print('--- TRANSAKSI PERTAMA ---');
  var rekeningEsti = RekeningBank('2514-1003-10', 7500000);
  rekeningEsti.cekSaldo();
  
  print('\n--- MELAKUKAN PENYETORAN ---');
  rekeningEsti.deposit(2500000);
  rekeningEsti.cekSaldo();
  
  print('\n--- MELAKUKAN PENARIKAN ---');
  rekeningEsti.tarik(1000000);
  rekeningEsti.cekSaldo();
  
  print('\n--- TRANSAKSI KEDUA ---');
  var rekeningAnggraini = RekeningBank('2514-1003-31', 10000000);
  rekeningAnggraini.cekSaldo();
  
  print('\n--- MELAKUKAN PENARIKAN ---');
  rekeningAnggraini.tarik(5000000);
  rekeningAnggraini.cekSaldo();
  
  print('\n--- PERCOBAAN PENARIKAN GAGAL ---');
  rekeningAnggraini.tarik(20000000);
  rekeningAnggraini.cekSaldo();
}
